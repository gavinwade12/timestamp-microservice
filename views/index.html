Timestamp Microservice

Usage:

    Pass a string as a parameter, and it will check to see whether that string contains either a unix timestamp or a natural language date (example: January 1, 2016).
If it does, it returns both the Unix timestamp and the natural language form of that date. If it does not contain a date or Unix timestamp, it returns null for those properties.

Examples:

  • /December%2015,%202015
  • /1450137600000

Example output:

{ "unix": 1450137600000, "natural": "December 15, 2015" }
